2  La rasem a wãn poore, bãmb lebg n kẽnga Kapɛrnawum, la kibarã saagame tɩ nebã bãng tɩ b bee yiri.
2  Rẽ n so tɩ neb wʋsg tigim taab hal tɩ vɩʋʋg kae baa rignoorẽ wã ye. La a Zeezi sɩngame n taasd-b koɛɛgã.
3  B talla ned sẽn ki bogr n wa bãmb nengẽ. Rap a naas n zã-a.
4  La kʋʋngã yĩnga, b ra pa tõe n tall-a n ta a Zeezi nengẽ ye. Rẽ n so tɩ b rʋ roogã, n leb a Zeezi sẽn beẽ wã n kɩbg võor n sik gãag ning zug ki-bogrã sẽn gãe wã.
5  A Zeezi sẽn yã tɩ b tara tẽebã, b yeela ki-bogrã: “M biiga, fo paama f yel-wẽnã sugri.”
6  La gʋlsdb kẽer n da be be. B ra zĩime n tagsd b sũyẽ yaa:
7  “Bõe yĩng tɩ ra-kãngã gomd woto? Bãmb paoogda Wẽnnaam. Ãnd n tõe n kõ yel-wẽnã sugr tɩ sã n pa Wẽnnaam bala?”
8  La zĩig pʋgẽ, a Zeezi bãngame tɩ b tagsda woto. Rẽ n so tɩ b sok-b yaa: “Bõe yĩng tɩ yãmb tagsd woto y sũyẽ wã?
9  Bʋg n yaa nana n yɩɩd tɩ yeel ki-bogrã: ‘Fo paama f yel-wẽnã sugr bɩ,’ bɩ yeel tɩ ‘Yik n dɩk f gãagã n kẽne’?
10  La mam na n wilga yãmb tɩ ninsaal Biigã tara noor tɩ kõ yel-wẽnã sugr tẽngã zugu.” Bãmb yeela ki-bogrã yaa:
11  “Mam yeta foom: Yik n dɩk f gãagã n kuil f yiri.”
12  Dẽ, a yikame n yɩ tao-tao n dɩk a gãagã n kẽn b fãa taoor n yi. Rẽ linga b fãa wʋsgo, la b ra pẽgda Wẽnnaam n yetẽ: “Tõnd zɩ n yã bõn-kãngã buud ye.” 
13  Bãmb le yiime n kẽng mogrã noore, la kʋʋngã fãa ra ket n wata bãmb nengẽ, tɩ b sɩng b zãmsgo.
14  Bãmb sẽn wa n loogdẽ, b yãa a Alfe biig a Levi t’a zĩ yõor-yaood reegr zĩigẽ. La bãmb yeela yẽ yaa: “Wa tũ maam.” Zĩig pʋgẽ, yẽ yikame n pʋg bãmba.
15  Rẽ poore, a Zeezi sẽn wa n zĩ a Levi roogẽ wã n dɩtẽ, yõor-yaood dɛɛgdb la yel-wẽn-maandb wʋsg n naag bãmb ne b karen-biisã n dɩtẽ, bala b wʋsg ra pʋgda bãmba.
16  La Fariziẽ-rãmbã gʋlsdb sẽn yã bãmb sẽn dɩt ne yel-wẽn-maandbã la yõor-yaood dɛɛgdbã, b sɩngame n yet bãmb karen-biisã yaa: “Bãmb dɩta ne yõor-yaood dɛɛgdbã la yel-wẽn-maandbã la?”
17  A Zeezi sẽn wʋm dẽ, b yeel-b lame: “Pa nin-kɛgemsã n baood tɩp ye. Yaa bãad-rãmb n baood tɩpa. Mam pa wa n na n bool nin-tɩrs ye. M waame n na n bool yel-wẽn-maandba.” 
18  La a Zã karen-biisã ne Fariziẽ-rãmbã ra minim n loeta noore. Rẽ n so tɩ b kolg n sok a Zeezi yaa: “Bõe yĩng t’a Zã karen-biisã ne Fariziẽ-rãmbã karen-biisã minim n loet noore, tɩ yãmb karen-biisã yaool n pa loet noore?”
19  La a Zeezi leoka bãmb n yeele: “Pʋg-paal sɩd sã n ket n be ne a zo-rãmba, b tara bũmb b sẽn tõe n tik n loe noor sɩda? Pʋg-paal sɩdã sã n be ne-b fãa, b pa tõe n loe noor ye.
20  La wakat n wat tɩ b na n yiis pʋg-paalã sɩd b sʋka. Rẽ, b na yaool n loe noore.
21  Ned pa be sẽn na n wãag peen-paall n lisg fu-kʋdr ye. A sã n maan woto, peen-paallã na n yõbgame n kɩt tɩ fu-kʋdrã kẽeg n yɩɩg pĩndã.
22  Sẽn paase, ned pa na n sui divẽ-paalg wao-kʋdr pʋgẽ ye. A sã n maan dẽ, divẽ wã na n kɩtame tɩ waoogã pusgi, tɩ divẽ wã ne waoogã fãa yɩ bõne. La nebã suiida divẽ-paalg wao-paalg pʋgẽ.” 
23  A Zeezi wa n pasgda pʋt Vʋʋsg daare, la b karen-biisã sɩngame n fãrd ka-zutã la b kẽnda.
24  Rẽ la Fariziẽ-rãmbã yeel a Zeezi yaa: “Ges-y-yã! Bõe yĩng tɩ b maand bũmb sẽn pa segd n maan Vʋʋsg daare?”
25  La bãmb yeel-b lame: “Yãmb zɩ n karem bũmb ning a Davɩɩd sẽn maane, rɩɩb sẽn wa n paoog-a tɩ kom tar yẽ ne rap nins sẽn da be ne-a wã la?
26  Maan-kʋʋdbã taoor soab a Abiataar kibarã pʋgẽ, a Davɩɩd sẽn yɩ wãn n kẽ Wẽnnaam doogẽ wã, la a sẽn wãb burã b sẽn dɩk n kõ Wẽnnaamã,* la a yãk n kõ rap nins sẽn da be ne yẽ wã, bũmb sẽn yaa maan-kʋʋdbã bal n tar sor n na n wãbe, tɩ ned baa a ye pa segd n wãbã la?”
27  Rẽ poore, a Zeezi yeel-b lame: “Vʋʋsg daarã sẽn be wã yaa ninsaal yĩnga, la ninsaal sẽn be wã pa Vʋʋsg daarã yĩng ye.
28  Woto, ninsaal Biigã yaa Vʋʋsg daarã Zu-soab me.”
